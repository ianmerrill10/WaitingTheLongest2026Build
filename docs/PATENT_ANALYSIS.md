# PATENT ANALYSIS — WaitingTheLongest.com
## Research Date: June 2026
## Conclusion: DO NOT FILE

---

## Prior Art Found

**US Patent Application 20150046354A1** — "System and method for matching potential adopters with animals in shelters"

Filed by PetPoint (shelter management software company). This patent explicitly covers:
- Sorting adoptable animals by duration in shelter
- FIFO time-based ranking of shelter animals
- Matching adopters with longest-waiting animals

**What this means:** The core concept of WaitingTheLongest — ranking dogs by wait time — has documented prior art. A utility patent on this feature would be rejected.

---

## Alice Doctrine Analysis

**35 U.S.C. § 101 — Patent Eligibility**

Under *Alice Corp. v. CLS Bank International* (2014), software patents must clear a two-step test:

1. **Step 1:** Is the claim directed to an abstract idea? → YES. Sorting by time is an abstract idea.
2. **Step 2A:** Does the claim add "significantly more" beyond the abstract idea? → NO. Sorting shelter records by intake date using a database is routine and conventional.

**Verdict:** WaitingTheLongest's sort-by-wait-time feature fails Step 1 of Alice. Any utility patent claim would almost certainly be invalidated in litigation.

---

## Design Patent Possibility

A design patent (protecting visual appearance only) is technically possible:
- Cost: ~$1,500–3,000 (USPTO + attorney fees)
- Protects: The specific visual design of the wait counter / ranking display
- Does NOT protect: The concept, the algorithm, the data model, or any functional aspect
- Enforceability: Low — any competitor changes the layout and the patent is circumvented

**Verdict:** Not worth pursuing for a startup. Visual design can be copied by changing colors and layout. The moat is brand, data, and network effects — not design.

---

## Trademark

The name **"WaitingTheLongest"** is trademark-eligible if used in commerce:
- Classes: IC 042 (computer services, online platform for animal adoption)
- Cost: ~$250–350 per class (USPTO TEAS Plus)
- Timeline: 8–12 months for registration
- Value: Protects the brand name from copycats. Worth doing once revenue is established.

**Recommendation:** Register the trademark once the site launches and generates even $1 in revenue. Not a priority for MVP.

---

## What Actually Protects WaitingTheLongest

| Moat | How to Build It |
|------|----------------|
| **Data network effects** | More shelters → more dogs → more users → more pressure on shelters to join |
| **Brand authority** | Be the recognized name for "longest-waiting dogs." PR, TikTok, media coverage. |
| **SEO dominance** | Own "longest waiting shelter dogs" SERPs — hard to dislodge once established |
| **Relationship lock-in** | Shelter integrations, API partnerships, breed-specific rescue networks |
| **First mover** | No direct competitor exists at this scale. Launch fast. |

---

## Recommendation

**DO NOT file any patent.** The utility patent will fail Alice. The design patent costs money and protects almost nothing. The trademark is worthwhile later but not now.

**Spend the $5,000–10,000 on:**
- Building the platform (done)
- Marketing and TikTok content ($0 organic)
- Shelter outreach and API partnerships
- SEO content strategy

The best defense against copying is being so far ahead that copying doesn't matter.
